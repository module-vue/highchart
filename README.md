
1、安装依赖   npm install

2、根据需要执行相应命令

npm run serve-dev              开启开发环境服务

npm run serve-test             开启测试环境服务

npm run serve-prod             开启生产环境服务

npm run build-dev              打包开发环境代码

npm run build-test             打包测试环境代码

npm run build-prod             打包生产环境代码
